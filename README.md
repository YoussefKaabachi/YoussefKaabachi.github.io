# Portfolio · Youssef Kaabachi

Persönliche Portfolio-Website mit den Projekten **Echolot** und **TULatte**.

## Lokal öffnen

Die Website benötigt keinen Build-Schritt. `index.html` kann direkt im Browser geöffnet oder mit einem lokalen Server gestartet werden:

```bash
python -m http.server 8080
```

Danach `http://localhost:8080` öffnen.

## GitHub Pages

Dieses Verzeichnis ist als eigenständiges Repository vorgesehen. Nach dem Hochladen kann unter **Settings → Pages** die Veröffentlichung aus dem `main`-Branch und dem Ordner `/ (root)` aktiviert werden.
